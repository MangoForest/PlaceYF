# PlaceYF
一个辅助装修工具

A tool in ffxiv to help to place furniture.

### 怎么使用
解压缩后，以管理员身份运行 PlaceYF.exe

需要在旋转状态选中家具后使用

### 开发环境
#### 编译器
MSYS2 下的 MinGW-w64
#### 依赖
Nana, gRPC
